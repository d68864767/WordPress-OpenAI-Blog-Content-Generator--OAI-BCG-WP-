# WordPress OpenAI Blog Content Generator (OAI-BCG-WP)

This WordPress plugin generates unique and SEO-optimized blog content based on user-defined topics and keywords.

## Description

Given the topic and keywords, this plugin generates a well-structured, SEO-optimized blog post for a WordPress site. The post includes an introduction, body, and conclusion. The plugin uses the keywords naturally throughout the post to enhance SEO. The language is engaging and informative to retain reader interest. The blog post is proofread for any grammatical errors and ensures it is coherent and meaningful.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/oai-bcg-wp` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->OAI-BCG-WP screen to configure the plugin.

## Usage

1. Go to the OAI-BCG-WP settings page.
2. Input your desired blog post title in the "Input Title" field.
3. Choose the length of your blog post: Short, Medium, or Long.
4. Click "Generate" to create your blog post.

## Files

- `oai-bcg-wp.php`: Main plugin file.
- `admin-settings.php`: Handles the admin settings page.
- `post-generator.php`: Generates the blog post based on user inputs.
- `seo-optimizer.php`: Optimizes the generated blog post for SEO.
- `proofreader.php`: Proofreads the generated blog post for grammatical errors.
- `styles.css`: Contains all the styles used in the plugin.
- `scripts.js`: Contains all the JavaScript used in the plugin.
